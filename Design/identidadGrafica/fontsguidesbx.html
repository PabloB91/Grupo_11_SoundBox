Guía tipográfica del sitio web

Esta guía tipográfica nos facilitará elegir los encabezados/titulares para las diferentes secciones del sitio web SOUNDBOX

H1: Bebas Neue Regular 400 en 48 px

Para títulos en banners y otras aplicaciones publicitarias del sitio web (Ej. el lema)

H2: Bebas Neue Regular 400 entre 28 - 36 px

Para títulos en banners y otras aplicaciones publicitarias del sitio web (Ej. el lema)

H3: Montserrat Bold 700 en 28 px

Para títulos en banners, aplicaciones publicitarias, noticias, blogs, etc. del sitio web (Ej. "SOUNDBOX llega a Argentina y Colombia")

H4: Montserrat SemiBold 600 en 24 px

Para subtítulos y textos cortos en banners, aplicaciones publicitarias, noticias, blogs, etc. del sitio web (Ej. "SOUNDBOX sigue llevando buena vibra a los amantes de la música en Argentina y Colombia")

H5: Montserrat Regular 400 en 21 px

Para subtítulos y cuerpos de texto cortos en las diferentes páginas, secciones , banners, aplicaciones publicitarias, noticias, blogs, etc. del sitio web (Ej. para resaltar detalles importantes de los productos musicales de este market place)

H6: Montserrat Regular 400 en 18 px

Para subtítulos y cuerpos de texto cortos en las diferentes páginas, secciones, banners, aplicaciones publicitarias, noticias, blogs, etc. del sitio web (Ej. para resaltar detalles importantes de los productos musicales de este market place)

Montserrat Regular 400 entre 14 - 16 px

Para cuerpos de texto y menús de navegación en las diferentes páginas, secciones, banners, aplicaciones publicitarias, noticias, blogs, etc. del sitio web (Ej. el menú principal, sub-menús, descripciones generales de productos, descripciones en cajita de compras, formularios, etc.)

Nota importante

Si bien en esta guía se plantean los lineamientos tipográficos en las etiquetas de los encabezados HTML, estos pueden aplicarse sobre cualquier selector específico que lo requiera (por ejemplo en clases que afecten articles, sections, divs, entre otros). Lo importante es tener una noción clara y coherente en la aplicación tipográfica del sitio web de lo macro a lo micro.
